Question 2:Explain inline , internal and external stylesheet.
answer := Inline styles are applied directly to HTML elements using the "style" attribute. Internal styles are defined within a style tag in the head section of the HTML document. External styles are written in separate CSS files and linked to the HTML document using the link tag.

Question 3: Justify-content allows you to do what?
answer := Justify-content is a CSS property used in flexbox and grid layouts to align and distribute space between items along the main axis of the container.

Question 4: Difference between absolute and relative positioning
answer := Absolute positioning removes an element from the normal document flow and positions it relative to its closest positioned ancestor (i.e., an ancestor with a position other than static). In contrast, relative positioning keeps the element in the document flow and positions it relative to its original position.

Question 5: What is the z-index in CSS?
answer := The z-index is a CSS property that determines the stacking order of positioned elements (i.e., elements with a position value other than static). Elements with a higher z-index value are stacked on top of elements with a lower z-index value. This property is particularly useful for controlling the visibility of overlapping elements.

Question 6: What is the difference between padding and margin?
answer := Padding is the space between the content of an element and its border, while margin is the space outside the border of an element. Padding creates space inside an element, whereas margin creates space outside of it.
